# Challenge1
Desafío Latam challenge número 1 - Data Science
Este proyecto es un desafío para **realizar recomendaciones a don Juan/Joao**.  
El objetivo es ayudar a decidir qué tienda de su cadena Alura Store debe vender para iniciar un nuevo emprendimiento. 
 
El objetivo es identificar la tienda menos eficiente y presentar una recomendación final basada en los datos.

## Estructura

1) Cargar y manipular bases de datos de las tiendas con la biblioteca Pandas.
2) Crear visualizaciones de los datos de ingresos, rendimientos de ventas, reseñas, etc. con la biblioteca Matplotlib.
3) Realizar recomendaciones al sr. Juan/Joao

## 📌 Objetivo
Este desafío permitirá:
1. **Analizar el desempeño de cada tienda**.
2. Aportar en la toma de decisión del sr. Juan sobre **Vender la "peor" tienda** basado en nuestra recomendación.
3. Mostrar un informe con los aspectos más destacados.
